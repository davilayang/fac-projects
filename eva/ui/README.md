# EVA — Event Voice Agent

Web UI for EVA, a voice agent built on LiveKit. Allows users to sign in and interact with the agent through the browser.

## Stack

- React 19, TypeScript, Vite
- Component library under `src/libs/components`, importable via `@libs/components`
- Icons use a shared `IconSize` type (`small`, `medium`, `large`)

## Development

```bash
pnpm install
pnpm dev
```
